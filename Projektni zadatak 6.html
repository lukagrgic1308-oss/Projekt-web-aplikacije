Projekt Slike by Luka Grgić

Objavljeno: 3. lipnja 2025.

U ovom projektu "Slike by Luka" se pokušava kreirati web stranica koja će omogućiti upload slika te dodjeljivanje atributa prema sadržaju koji se nalazi na slikama.

Neki od alata koji će se koristiti u ovom projektu su Visual basic koji služi kao osnovni program za kreiranje web stranice u HTML5 programskom jeziku.

Korisnik može podići sliku na web stranicu te joj dodjeliti "atribute" prema kojima se kasnije može pretražiti ta slika i sve slike koje imaju isti/e takve atribute.

Kao baza se koristi HTML5 programski jezik koji služi za kreiranje same web stranice.

Kako bi imali što bolju predodžbu molim pogledajte slike ispod.
